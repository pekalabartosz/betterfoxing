  1. Go to about:config in Firefox search bar and accept the warning if you'll get one.
  2. Search for toolkit.legacyUserProfileCustomizations.stylesheets and enable it by double clicking.
  3. Go to about:profiles and search for the profile you're currently using. Click "Show in Finder" (Root directory), open the catalog and create a new folder here named "chrome" if you don't have one.
  4. Download code from BetterFox and paste the userContent.css in chrome catalog.

Important: to add next theme, copy the code from it and paste to the exsisting userContent.css in the chrome catalog.
